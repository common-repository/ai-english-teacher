This plugin uses OpenAI to correct English grammar and rephrase sentences on your website.

== Description ==

The AI English Teacher plugin is an advanced tool designed to help website administrators improve the quality of their website's content. It utilizes OpenAI's advanced natural language processing algorithms to correct grammar and rephrase sentences with ease. 

[youtube https://www.youtube.com/watch?v=TRwzP5gUTmQ]

With the AI English Teacher plugin, website administrators can easily correct grammar mistakes and rephrase sentences to enhance the readability and clarity of their content. The plugin allows administrators to make changes and ensure quality content. 

To use the plugin, administrators simply highlight the text they wish to analyze, and click the "AI English Teacher" icon to activate the correction and rephrasing tools. The plugin then analyzes the text and provides suggestions for corrections, allowing website administrators to make changes quickly and easily.

== Installation ==

1. Download and unzip the plugin. Upload the unzipped folder to the wp-contents/plugins folder of your WordPress installation.
2. Active the plugin from the WordPress Plugins administration page.
3. OR, Go to WP admin panel, click 'Plugins' -> 'Add new'. In the search input box, type 'AI English Teacher'.
4. Install and activate the plugin.
